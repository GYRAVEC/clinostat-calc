# clinostat-calc

2軸クリノスタット (内側 r = 8.5 cm / 外側 r = 14.5 cm) の **カスタム重力プロファイル** 計算 CLI。
サンプルは 2軸の交点に置き、**遠心力を使わない時間平均方式** で目標 g を作る。

目標 g を渡すと、各フレームの回転速度（と必要なら外軸傾斜角）を返す。細菌培養での沈殿問題に対応した設計。

## 2つのモード

### Tilt モード (既定, **0 ≤ g_target ≤ 1g**)

外軸を鉛直から ψ = arccos(g_target) 傾けて運用する。

```
細胞が短時間平均で知覚する重力:  |g| = g0 · cos ψ
```

- 外側 ω_out: 傾斜軸まわりに高速回転 (鉛直成分以外を均す)
- 内側 ω_in: 超低速 (位相ドリフト周期 T_drift で重力ベクトル方向を1周)
- ハード変更が必要（傾斜マウント）だが **0〜1g 全域を1組の (ω_out, ω_in) でカバー**

ψ の例:

| g_target          | ψ (鉛直から) |
|-------------------|--------------|
| 1.0g              | 0°          |
| 0.8g              | 36.87°       |
| 0.5g              | 60°         |
| 0.378g (火星)     | 67.79°       |
| 0.166g (月)       | 80.45°       |
| 0g (微小重力)     | 90° (水平)  |

### Switching モード (`--mode switching`, **0 ≤ g_target ≤ 0.5g**)

傾斜なしで運用したい場合の従来方式。Mode A (1:1+位相ドリフト, 0.5g) と Mode B (黄金比, 0g) を時間配分:

```
g_target = 0.5 · τ_A    ⇒   τ_A = 2 · g_target
```

- ハード変更不要
- 上限 0.5g

## なぜ沈殿しないか

3つのタイムスケールを分離:

```
T_outer (1回転 ~12秒) ≪ T_bio (細胞応答 ~10分) ≪ T_drift (~20分) ≪ T_sed (~3時間)
```

- 細胞は短時間平均として g_target を **知覚** (生物応答時間より早く方向は変わらない)
- T_drift で重力方向が1周するので **長時間平均では蓄積方向ゼロ**

既定値 (`--base-rpm 5 --drift-min 20`) でバクテリア (v_sed ≈ 1 μm/s, チャンバ半径 5 mm) なら、
火星 (0.378g) のドリフト振幅は 0.14 mm (チャンバの 2.9%) で十分安全。

## 使い方

```bash
# 既定: tilt モード
python3 clinostat.py 0.378           # 火星
python3 clinostat.py mars            # プリセット
python3 clinostat.py 0.8             # 0.8g (高g例)
python3 clinostat.py "3.72 m/s2"     # m/s² 単位

# Switching モード (傾斜なし運用)
python3 clinostat.py 0.166 --mode switching

# 両方並べて比較
python3 clinostat.py 0.378 --mode both

# 全プリセット一括
python3 clinostat.py --all

# 対話モード
python3 clinostat.py                 # 既定 tilt
python3 clinostat.py --mode switching
```

### フラグ

| フラグ              | 既定値    | 意味                                              |
|---------------------|-----------|---------------------------------------------------|
| `--mode`            | `tilt`    | `tilt` / `switching` / `both`                     |
| `--base-rpm`        | 5.0       | 基準回転速度 [RPM]                                |
| `--drift-min`       | 20.0      | 位相ドリフト周期 [min]                            |
| `--cycle-min`       | 60.0      | switching モードの切替周期 [min]                  |
| `--chamber-mm`      | 5.0       | チャンバ半径 [mm] (沈降診断用)                    |
| `--no-color`        | -         | ANSI カラー無効化                                 |

### 対話モードのコマンド

| 入力                  | 動作                                |
|-----------------------|-------------------------------------|
| `0.166` / `mars`      | 計算実行                            |
| `3.72 m/s2`           | m/s² 単位入力                       |
| `list` / `presets`    | プリセット一覧                      |
| `h` / `help` / `?`    | プリセット名表示                    |
| `q` / `quit` / `exit` | 終了                                |

## プリセット

`earth, mars, moon, ceres, europa, titan, iss, ug`

## 出力フィールド

### Tilt モード
- **ψ**: 外軸を鉛直から傾ける角度 (ハードウェア設定値)
- **ω_out**: 傾斜軸周りの高速回転 (鉛直以外の重力成分を均す)
- **ω_in**: 重力ベクトル方向を T_drift 周期で1周させる低速回転

### Switching モード
- **τ_A**: Mode A の時間配分 = 2·g_target
- **Mode A の ω_in, ω_out**: ほぼ同速、差 ε = 2π/T_drift
- **Mode B の ω_in, ω_out**: 比 = 1/φ (黄金比)

両モード共通:
- **ドリフト振幅**: 細胞が描く軌跡の最大半径。チャンバ半径より小さければ沈殿なし

## モード選択ガイド

| 状況                                    | 推奨            |
|----------------------------------------|-----------------|
| g_target > 0.5g                        | tilt 一択       |
| ハードウェアを傾斜できる                  | tilt           |
| 傾斜マウントを作れない                   | switching      |
| 既存装置 + 0.5g以下で済む              | switching      |
| 両方検討したい                          | `--mode both`  |
