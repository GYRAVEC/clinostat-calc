# clinostat-calc

2軸クリノスタット (内側 r = 8.5 cm / 外側 r = 14.5 cm) の **カスタム重力プロファイル** 計算 CLI。
目標重力 g_target を渡すと、各フレームの回転速度・必要なら傾斜角と、沈殿予測を返す。

実験目的に合わせて 3 つのモードを選べる。

## 3つのモード早見表

| モード | サンプル位置 | 重力源 | 沈殿 | 主な用途 |
|--------|------------|--------|------|---------|
| **tilt** (既定) | 2軸の交点 (r≈0) | 地球重力の射影 g₀·cos ψ | なし | 懸濁状態で培養したい |
| **centrifuge** | 内側4角 (r=8.5cm) | 内側回転の遠心力 ω²r | あり (火星/月と同じ) | Mars/Moon analog 実験 |
| **switching** | 2軸の交点 (r≈0) | 1:1 同期と黄金比の時間配分 | なし | 傾斜不可で 0–0.5g |

詳細な原理は [THEORY.md](THEORY.md) を参照。

## Tilt モード — 懸濁培養向け (既定)

外軸を鉛直から ψ = arccos(g_target) 傾けて、外側を高速、内側を超低速で回す。

```
細胞が知覚する重力 |g| = g0·cos ψ   (短時間平均)
細胞の長時間平均 ⟨g⟩ = 0          (位相ドリフトで方向が一周し蓄積ゼロ)
```

| g_target | 傾斜角 ψ | 沈殿振幅 (5mm チャンバ仮定) |
|----------|---------|----------------------------|
| 1.0 g | 0° (傾斜なし) | (回転不要、静置) |
| 0.5 g | 60° | 0.19 mm (3.8%) |
| 0.378 g (火星) | 67.79° | 0.14 mm (2.9%) |
| 0.166 g (月) | 80.44° | 0.06 mm (1.3%) |
| 0g (微小重力) | 90° (水平) | 0.004 mm (0.1%) |

→ 0〜1g 全域を 1組の (ω_out, ω_in) でカバー、沈殿なし。

## Centrifuge モード — Mars/Moon analog 実験向け

サンプルを内側フレーム4角 (内軸から r = 8.5cm) に置き、内側回転の遠心力で g_target を負荷。

```
ω_in² × r = g_target × g0   ⇒   ω_in = √(g_target × g0 / r)
```

| g_target | ω_in 内側 | ω_out 外側 | 沈降速度 | 24h培養での壁滞在 |
|----------|-----------|-----------|---------|------------------|
| 1.0 g | 102.6 RPM | 5 RPM | 1.00 μm/s | 94% (= 地球そのもの) |
| 0.378 g (火星) | 63.1 RPM | 5 RPM | 0.378 μm/s | 85% |
| 0.166 g (月) | 41.8 RPM | 5 RPM | 0.166 μm/s | 65% |
| 0.01 g | 10.3 RPM | 5 RPM | 0.010 μm/s | 0% (壁未到達) |

→ 部分重力 + 沈殿の両方を再現。火星表面で起きる現象をそのまま地上で再現したい場合に使う。
→ 細胞は遠心方向に沈み、容器壁に集まる (これは原理的に避けられない。tilt モードと使い分け)。

## Switching モード — 傾斜不可の運用向け (上限 0.5g)

```
g_target = 0.5 · τ_A    ⇒   τ_A = 2 · g_target
```

外側を 5 RPM 固定、内側を Mode A (1:1+ε、0.5g) と Mode B (黄金比、0g) で時間配分。
傾斜マウントが作れない場合の代替。0.5g までしかカバーできないが沈殿はなし。

## なぜ沈殿しないか (tilt / switching)

3つのタイムスケールを分離:

```
T_outer (1回転 ~12秒) ≪ T_bio (細胞応答 ~10分) ≪ T_drift (~20分) ≪ T_sed (~3時間)
```

- 細胞は短時間平均として g_target を **知覚** (生物応答時間より早く方向は変わらない)
- T_drift で重力方向が1周するので **長時間平均では蓄積方向ゼロ**

既定値 (`--base-rpm 5 --drift-min 20`) でバクテリア (v_sed ≈ 1 μm/s, チャンバ半径 5 mm) なら、
火星 (0.378g) のドリフト振幅は 0.14 mm (チャンバの 2.9%) で安全。24-48h 培養でも合計変位 < 40%。

## 使い方

```bash
# tilt モード (既定)
python3 clinostat.py 0.378           # 火星
python3 clinostat.py mars            # プリセット
python3 clinostat.py 0.8             # 0.8g (高g例)
python3 clinostat.py "3.72 m/s2"     # m/s² 単位

# centrifuge モード (Mars analog)
python3 clinostat.py 0.378 --mode centrifuge
python3 clinostat.py 0.378 --mode centrifuge --culture-hours 48

# switching モード (傾斜不可運用)
python3 clinostat.py 0.166 --mode switching

# 3モード並べて比較
python3 clinostat.py 0.378 --mode all

# 全プリセット一括
python3 clinostat.py --all

# 対話モード
python3 clinostat.py
python3 clinostat.py --mode switching
```

### フラグ

| フラグ              | 既定値    | 意味                                              |
|---------------------|-----------|---------------------------------------------------|
| `--mode`            | `tilt`    | `tilt` / `centrifuge` / `switching` / `all`       |
| `--base-rpm`        | 5.0       | 基準回転速度 [RPM]                                |
| `--drift-min`       | 20.0      | 位相ドリフト周期 [min] (tilt/switching)           |
| `--cycle-min`       | 60.0      | switching モードの切替周期 [min]                  |
| `--chamber-mm`      | 5.0       | チャンバ半径 [mm] (沈降診断用)                    |
| `--sample-r-cm`     | 8.5       | centrifuge モード時のサンプル内軸からの半径 [cm]  |
| `--culture-hours`   | 24        | centrifuge モードの沈降診断用の培養時間 [h]       |
| `--no-color`        | -         | ANSI カラー無効化                                 |

### 対話モードのコマンド

| 入力                  | 動作                                |
|-----------------------|-------------------------------------|
| `0.166` / `mars`      | 計算実行                            |
| `3.72 m/s2`           | m/s² 単位入力                       |
| `list` / `presets`    | プリセット一覧                      |
| `h` / `help` / `?`    | プリセット名表示                    |
| `q` / `quit` / `exit` | 終了                                |

## プリセット

`earth, mars, moon, ceres, europa, titan, iss, ug`

## モード選択ガイド

| 状況                                                       | 推奨            |
|------------------------------------------------------------|-----------------|
| 火星/月の表面環境を細胞に再現したい (Mars analog)              | centrifuge      |
| 部分重力下で細胞を懸濁状態で観察したい                        | tilt           |
| g_target > 1.0g (過重力)                                    | centrifuge      |
| g_target > 0.5g (傾斜可能な装置がある)                       | tilt           |
| g_target > 0.5g (傾斜できない、過重力NG)                     | (該当なし、ハード見直し) |
| ハードウェアを傾斜できない、0.5g以下でよい                    | switching       |
| 既存装置 + 0.5g以下で済む、傾斜したくない                    | switching       |
| どれが合うか比較したい                                       | `--mode all`   |

## 注意

- **tilt モード**: サンプルを 2軸交点 (中央マウント) に置く必要あり。内側フレーム 4角は使わない。フレーム半径 8.5/14.5cm は重力計算に登場せず、構造寸法としてのみ意味を持つ。
- **centrifuge モード**: サンプルを内側フレーム 4角に置く。内側フレーム半径 8.5cm が遠心力 ω²r の式に直接登場する。
- 各モードの理論的妥当性と沈殿挙動の式は [THEORY.md](THEORY.md) を参照。
