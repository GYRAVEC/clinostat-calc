# clinostat-calc

2軸クリノスタット (内側 r = 8.5 cm / 外側 r = 14.5 cm) の **部分重力プロファイル** を計算する CLI ツール。
細菌培養での沈殿問題を避けるため、サンプルを 2軸の交点に置く構成を前提に、**遠心力を使わない時間平均方式** で目標 g を作る。

## 原理

2軸ジンバル (外軸=水平x̂、内軸=ŷ初期) でサンプル系に現れる重力ベクトル:

```
g_sample(t) = g0 · ( cos α · sin β,  −sin α,  −cos α · cos β )
             α = ω_out · t,   β = ω_in · t
```

ほぼ全ての ω 比で時間平均は 0 (微小重力) になるが、**ω_in = ω_out (1:1 完全同期)** の場合のみ:

```
⟨g_sample⟩ = g0/2 · ( sin δ, 0, −cos δ ),   |⟨g⟩| = 0.5g  (位相 δ に依らず)
```

これを使い、**Mode A** (1:1+微小位相ドリフト → 0.5g) と **Mode B** (黄金比 → 0g) を時間配分する:

```
g_target = 0.5 · τ_A + 0 · (1 − τ_A)    ⇒    τ_A = 2 · g_target
```

- 火星 0.378g → Mode A 75.6% / Mode B 24.4%
- 月 0.166g → Mode A 33.2% / Mode B 66.8%
- 微小重力 → Mode B 100%

**達成可能範囲: 0 ≤ g_target ≤ 0.5g**（1g 静止は沈殿が出るため除外）

## なぜ沈殿しないか

Mode A の 1:1 で位相を ε で微小ドリフトさせると、0.5g ベクトル方向がチャンバ内をゆっくり一周する。
時間スケールの分離:

```
T_rotation ≪ T_bio (細胞応答 ~10 min) ≪ T_drift (~20 min) ≪ T_sed (~3 h)
```

- 細胞は短時間平均として 0.5g を **知覚**
- ドリフトで方向が回るので **長時間平均では沈殿しない**

既定値 (`--base-rpm 5 --drift-min 20`) でバクテリア (v_sed ≈ 1 μm/s, チャンバ半径 5 mm) ならドリフト振幅は約 0.2 mm (チャンバの 3.8%)。

## 使い方

```bash
# 単発: g 単位
python3 clinostat.py 0.378

# プリセット
python3 clinostat.py mars

# m/s²単位
python3 clinostat.py "3.72 m/s2"

# 全プリセット一括
python3 clinostat.py --all

# 対話モード
python3 clinostat.py
```

### 主なフラグ

| フラグ          | 既定値 | 意味                                            |
|-----------------|--------|------------------------------------------------|
| `--base-rpm`    | 5.0    | 基準回転速度 [RPM]                              |
| `--drift-min`   | 20.0   | Mode A 位相ドリフト周期 [min]                   |
| `--cycle-min`   | 60.0   | モード A↔B 切替サイクル [min]                  |
| `--chamber-mm`  | 5.0    | チャンバ半径 [mm] (沈降診断用)                 |
| `--no-color`    | -      | ANSI カラー出力を無効化                         |

### 対話モードのコマンド

| 入力                    | 動作                                  |
|-------------------------|--------------------------------------|
| `0.166` / `mars`        | プロファイル計算                      |
| `3.72 m/s2`             | m/s² 単位で入力                       |
| `list` / `presets`      | プリセット一覧                        |
| `h` / `help` / `?`      | プリセット名簡易表示                  |
| `q` / `quit` / `exit`   | 終了                                  |

## プリセット

`mars, moon, ceres, europa, titan, iss, ug` (earth=1g は範囲外なので除外)

## 出力フィールドの意味

- **τ_A**: Mode A の時間配分。`2 · g_target`
- **Mode A の ω_in / ω_out**: ほぼ同速、差 ε = 2π/T_drift で位相ドリフト
- **Mode B の ω_in / ω_out**: 比 = 1/φ (黄金比) で時間平均 → 0g
- **ドリフト振幅**: Mode A 中の細胞が描く軌跡の最大半径。チャンバ半径より十分小さければ沈殿しない

## 注意

- 上限 0.5g は **等速 2軸+サンプル中心構成の物理限界** (1:1 モードでの時間平均値)。それ以上の g を作るには遠心成分の併用が必要で、その場合は沈殿が出る
- ドリフト周期 T_drift は細胞応答時間より長く取る必要 (短すぎると細胞は 0g として応答する)
