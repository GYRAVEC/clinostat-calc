# 2軸クリノスタット 部分重力シミュレーション動作原理レポート

このレポートは「2軸クリノスタットを使って、サンプルに狙いどおりの部分重力をかけられるのか、また細胞が沈殿してしまうのか」を、3つの運用モードについて物理の式と数値で確かめるものです。専門用語は登場するたびに普通の言葉で説明します。

---

## 1. このレポートで答えること

火星の重力 (地球の 0.378 倍) や月の重力 (0.166 倍) のような **「地球と異なる重力」をサンプル内の細胞に体験させたい**、ということを目的にしています。サンプルの置き方と回転のさせ方によって、4つの運用モードがあります。

| モード | サンプル位置 | 重力源 | 沈殿 | 用途 |
|--------|------------|--------|------|------|
| **Tilt** (傾斜) | 2軸の交点 (回転中心) | 地球重力の射影 g₀·cos ψ | なし | 懸濁状態で培養したい場合 |
| **Centrifuge** (遠心) | 内側フレーム4角 (r=8.5cm) | 内側回転の遠心力 ω²r | あり (火星/月と同じ性質) | Mars/Moon analog 実験 |
| **Hybrid** | **任意の r** (例: 2cm オフセット) | tilt と遠心の合成 (直交合成) | 主に なし (遠心由来の小累積あり) | 中央からわずかにずれたサンプル配置 |
| **Switching** (切替) | 2軸の交点 | 1:1 同期と黄金比の時間配分 | なし | 傾斜機構なしで運用したい場合 |

clinostat-calc ツールは、目標重力とモードを入力すると次を出力します。

- 必要なハードウェア設定 (Tiltなら傾斜角ψ、Centrifugeならサンプル半径)
- 外側フレームの回転速度
- 内側フレームの回転速度
- 沈殿予測 (発生するか・するならどれくらいの時間スケールか)

このレポートでは、各モードについて

(a) 細胞は確かに狙った重力を「感じる」のか
(b) 細胞は容器のどこかに集まって沈殿してしまわないか

を、計算と数値で確認します。前半 (§2〜§11) は Tilt モードを詳しく扱い、§12 で Centrifuge モード、§13 で各モードを比較します。

---

## 2. 装置の構成

2軸クリノスタットは入れ子の2つのフレームでできています。

```
            ←─── 外側フレームの回転軸 ───→
            (これを「外側軸」と呼びます)

         ┌────----─────────────────-─────┐
         │                               │
         │        ┌──────────────┐       │ ← 外側フレーム
         │        │              │       │
         │        │ ⊕            │       │ ← サンプル本体
         │        │  サンプル      │       │   (細胞培養液)
         │        │              │       │
         │        └──────────────┘ ← 内側フレーム
         │                               │
         └───────----───────-────────────┘
                         ↑
                       内側フレームの回転軸
                       (これを「内側軸」と呼びます)
                       外側軸に対して直角
```

| 部品 | 役割 | 半径 (このレポートでの値) |
|------|------|---------------------------|
| 外側フレーム | 外側軸まわりに高速回転 | 14.5 cm |
| 内側フレーム | 外側フレームの中に納まり、内側軸まわりに別速度で回転 | 8.5 cm |
| サンプル | 細胞培養液を入れる容器。**2つの軸が交差する点に置く** | チャンバ半径 5 mm を想定 |

**重要**: サンプルを 2つの軸の交点に置くというのが本レポートの大前提です。交点に置けば、サンプルは回転軸から離れていないので、遠心力 (回転で外に飛ばされる力) を一切受けません。重力以外に余計な力が入らない、きれいな状況になります。

## 3. 3つの座標系を頭の中で分ける

物理計算では「どこから見ているか」で結果が変わるので、3つの視点 (座標系) を区別します。

### 3.1 実験室系

実験室の床に固定された座標系。私たちが装置を眺めるときの目線です。

- 重力は常に真下を向きます。
- 装置が回ろうが何しようが、この座標系自体は動きません。

このレポートでは、地面に対する **上向きの方向を「上方向」**、**下向きを「下方向」** と呼びます。重力は実験室系で見ると常に下方向を向く、強さは 9.81 m/秒² です。

### 3.2 外側フレーム座標系

外側フレームと一緒に回転する座標系。外側フレームの上に乗って一緒に回っている観測者の目線です。

- 外側フレームから見ると、自分は止まっているように見えます。
- 代わりに、実験室や重力ベクトルがゆっくり (もしくは速く) 回っているように見えます。

### 3.3 サンプル座標系

サンプル本体と一緒に動く座標系。サンプルの中にいる細胞の目線です。サンプルは外側フレームの上の内側フレームの中に固定されているので、**外側と内側の両方の回転を継承して動きます**。

- 細胞が「上下左右」をどう感じるかは、すべてこのサンプル座標系で考えます。
- 細胞が重力をどの方向にどれくらい感じるか — それを知りたいのが本レポートの主題です。

## 4. 装置を傾けると何が起きるか (直感的説明)

外側軸を実験室の床に対して垂直 (=床と直角に立てる) から、徐々に倒していくことを考えます。倒し具合を「傾斜角」と呼び、ψ (ギリシャ文字のプサイ) で表記します。

![傾斜モードの幾何学: 傾斜角ψ=0°, 60°, 90° の3パネルで、外側軸の傾きと地球重力の関係を示す模式図](https://raw.githubusercontent.com/GYRAVEC/clinostat-calc/main/image/Gemini_Generated_Image_gpu3i4gpu3i4gpu3.png)

*図1. 傾斜角 ψ を変えたときの外側軸 (グレーの棒) と地球重力 (黒矢印 g₀) の位置関係。左: ψ=0° で外側軸が鉛直、重力は軸と平行。中央: ψ=60° で重力を軸方向 (g₀·cos ψ) と直角方向 (g₀·sin ψ) に分解。右: ψ=90° で外側軸が水平、重力は軸と直角。サンプル (黒点) は常に2軸の交点に位置する。*

地球重力は常に下を向いていますが、外側軸の向きが変わると、重力を「軸方向の成分」と「軸に直角な成分」に分けて考えたとき、それぞれの大きさが変わります。

- 外側軸が床と直角 (ψ = 0°) のとき: 重力は外側軸と平行 (下向き)。軸方向成分 = 重力そのもの (地球の 1.0 倍)。直角成分 = ゼロ。
- 外側軸が斜め (ψ = 60°) のとき: 重力は軸に対して 60° の角度。軸方向成分は cos 60° = 0.5 倍。直角成分は sin 60° = 0.866 倍。
- 外側軸が床と水平 (ψ = 90°) のとき: 重力は外側軸と直角。軸方向成分 = ゼロ。直角成分 = 重力そのもの。

ここで重要なのは:

- **軸方向の重力成分**は、外側軸まわりに回転しても向きが変わらない (回転軸と平行な方向は回しても回らない)。
- **軸に直角な重力成分**は、外側軸まわりに回転すると、ぐるぐる回ってしまう。

つまり、外側フレームを十分な速さで回し続けると:

- 軸方向成分は **そのまま残る** (時間平均しても変わらない、強さ g₀ × cos ψ)。
- 直角成分は **ぐるぐる回って、時間で見るとプラスとマイナスが相殺してゼロになる** (時間平均してゼロ)。

結局サンプルは、**「外側軸方向に向いた、強さ g₀ × cos ψ の重力」だけを感じる** ことになります。これが「傾斜モードで部分重力を作る」原理です。傾斜角 ψ を変えるだけで、g₀ × 1 (=地球と同じ) から g₀ × 0 (=無重力) まで、好きな強さを連続的に作れます。

## 5. 内側フレームの役目 (沈殿を防ぐ)

しかし上の説明だけでは細胞が沈殿してしまいます。なぜか?

サンプルから見ると「外側軸方向にある強さの重力」が **常に同じ方向に向いている** ので、細胞はその方向に向かってゆっくり沈み続けます。地球の 0.378 倍なので、地球上の 0.378 倍の速度で沈みますが、長時間たてばサンプル容器の壁にぶつかります。

これを防ぐのが内側フレームの役目です。**内側フレームをゆっくり回すと、サンプル座標系から見たときの「外側軸方向」がゆっくりサンプルの中で1周します**。

```
内側フレームを回す前: 重力は常に矢印 → の方向
                       ⊕→→→→→ → → 細胞は右に集まる

内側フレームを回した後: 重力の向きがゆっくり1周
                       ⊕↑→↓← サンプル中で一周
                       細胞は中心の周りをぐるぐる回って戻ってくる
                       → どこにも集まらない
```

ここで大事な時間スケールの分離があります。

- 内側フレームの回転は **十分ゆっくり** にしないといけません。なぜなら、細胞が「自分の感じている重力の強さ」を生物的に判断するのに時間がかかるからです (細菌で約10分)。もし内側フレームが速く回ると、細胞は重力の向きをずっと変えられて、平均すると「重力なし」と感じてしまい、狙いと違ってしまいます。
- 同時に **十分速く** でないといけません。なぜなら、内側フレームが遅すぎると、1周してくるまでに細胞が沈殿してしまうからです。

このバランスを取った設定が、後で出てくる「位相ドリフト周期」(20分くらい) です。

## 6. サンプル系での重力ベクトルの計算

ここから式で確認します。出てくる記号を最初にまとめます。

| 記号 | 意味 | 単位 |
|------|------|------|
| g₀ | 地球の重力加速度 | 9.81 m/秒² |
| ψ (プサイ) | 外側軸を床の鉛直方向から倒した傾斜角 | 度 (もしくはラジアン) |
| α (アルファ) | 外側フレームの回転角度 (時間とともに大きくなる) | ラジアン |
| β (ベータ) | 内側フレームの回転角度 (時間とともに大きくなる) | ラジアン |
| 外側角速度 ω_out | 外側フレームが 1秒で回る角度 | ラジアン/秒 (もしくは毎分回転数 RPM) |
| 内側角速度 ω_in | 内側フレームが 1秒で回る角度 | ラジアン/秒 |
| 時刻 t | 装置を動かし始めてからの時間 | 秒 |

α と t の関係は `α = ω_out × t`、β と t の関係は `β = ω_in × t` です。

### 6.1 実験室系での重力

実験室系では地球重力は単純に下向きの一定ベクトル:

```
実験室系での重力ベクトル = (横方向はゼロ、横方向はゼロ、下向きに 9.81 m/秒²)
```

### 6.2 外側フレーム座標系での重力

外側軸を傾斜角 ψ 傾けて外側フレームを回転させると、外側フレーム上の観測者から見ると、地球重力は次の3成分に分解できます。

- **外側軸に沿った成分**: 大きさ `g₀ × cos ψ`。向きは外側軸に沿った下向き。外側フレームが回っても変わらない (常に一定)。
- **外側軸に直角な成分の、横方向 (q方向)**: 大きさは `g₀ × sin ψ × sin α` で、時間 t とともに振動する。
- **外側軸に直角な成分の、上方向 (p方向)**: 大きさは `g₀ × sin ψ × cos α` で、時間 t とともに振動する。

(細かい添え字 p, q は単に「外側軸に対して直角な2つの直角方向」だと思って構いません)

### 6.3 サンプル座標系での重力

さらに内側フレームの回転 (角度 β) を加味すると、サンプル座標系での重力は次の3成分になります。

```
方向(1)の成分  =  g₀ × sin ψ × cos α
方向(2)の成分  = -g₀ × cos ψ × sin β  -  g₀ × sin ψ × sin α × cos β
方向(3)の成分  = -g₀ × cos ψ × cos β  +  g₀ × sin ψ × sin α × sin β
```

(方向(1)(2)(3) はサンプル座標系での 3つの直角な方向。具体的に「上」「右」「奥」等にどう対応するかは初期姿勢で決まるので、ここでは番号でいいです)

これがサンプルから見た重力ベクトルの瞬間値。時間ともに激しく変動する複雑なものですが、ここから「細胞が感じる平均的な重力」を計算します。

## 7. 細胞が感じる重力 (短時間で平均した値)

### 7.1 細胞は短時間平均を感じる

細胞は重力を感じてから生物的に応答するまで時間がかかります (細菌でだいたい 10分くらい)。なので、細胞は瞬間値ではなく **「10分くらいで平均した重力」を感じる** とみなせます。

10分間の間に:

- 外側フレームは 5 RPM (毎分5回転) で回っているとすると、50 回転する → 外側の回転角度 α は何周もする → cos α や sin α の平均値はゼロになる。
- 内側フレームは 0.05 RPM (毎分0.05回転) で回るとすると、0.5 回転する → 内側の回転角度 β はまだほぼ動いていない → cos β や sin β はほぼ定数 (10分間の平均値はそのときの値 β₀ にほぼ等しい)。

これを 6.3 の3成分に当てはめると、サンプル座標系での重力の短時間平均は次のようになります。

```
方向(1)の平均  ≈  g₀ × sin ψ × 0 = 0
方向(2)の平均  ≈ -g₀ × cos ψ × sin β₀  -  g₀ × sin ψ × 0 × cos β₀ = -g₀ × cos ψ × sin β₀
方向(3)の平均  ≈ -g₀ × cos ψ × cos β₀  +  g₀ × sin ψ × 0 × sin β₀ = -g₀ × cos ψ × cos β₀
```

### 7.2 重力ベクトルの大きさ

「ベクトルの大きさ」は 3成分を 2乗して足して平方根を取れば出ます。

```
大きさ = √( 0² + (g₀ cos ψ sin β₀)² + (g₀ cos ψ cos β₀)² )
       = √( g₀² cos²ψ × (sin²β₀ + cos²β₀) )
       = √( g₀² cos²ψ × 1 )           ← sin² + cos² は常に 1
       = g₀ × cos ψ
```

**結論**: 内側フレームの瞬時の角度 β₀ がどんな値でも、サンプル系での平均重力の大きさは **常に g₀ × cos ψ** になります。これは外側軸を ψ 傾けたときの「軸方向成分」と一致していて、§4 の直感どおりの結果です。

つまり ψ = arccos(目標重力 ÷ g₀) と選べば、細胞は短時間平均として **正確に目標重力を感じます**。

例: 火星の重力 0.378 × g₀ を作りたい → ψ = arccos(0.378) = 67.79°。実際にツール出力と一致します。

### 7.3 重力の向き

サンプル系での平均重力の向きは、方向(2) と方向(3) の平面内で、β₀ で決まる角度を向きます。β₀ は時間とともに `β₀ = ω_in × t` でゆっくり増えていくので、平均重力の向きはサンプル系の中をゆっくり1周します。1周にかかる時間が「位相ドリフト周期」で、20分くらいに設定します。

これが §5 で言った「内側フレームをゆっくり回すと重力の向きが1周する」の実体です。

## 8. 細胞は沈殿しないか (長時間で見たときの動き)

### 8.1 長時間平均はゼロ

7章では「10分くらいで平均」を考えました。次は「もっと長く、たとえば数時間で平均」した重力を考えます。

数時間あれば内側フレームも何周もするので、cos β や sin β も平均ゼロになります。すべての三角関数の平均がゼロになると、重力ベクトルの長時間平均は **ベクトルとしてゼロ** になります。

これが沈殿しない理由です。長時間で見ると、細胞は「上方向」も「下方向」もない状態に置かれているので、特定の方向に動き続けることがありません。

### 8.2 細胞は実際にどう動くか

細胞が重力の向きに沈む速度は、自由沈降速度 (細菌で約 1 μm/秒、つまり 1秒あたり 1マイクロメートル) に重力の強さの比 (g_target = 0.378 など) を掛けたものです。

10分間 (細胞応答時間) でどれくらい動くか:

```
変位 = 沈降速度 × 目標重力比 × 時間
     = 1 μm/秒 × 0.378 × 600秒
     = 227 μm = 0.227 mm
```

10分のあいだ、細胞はサンプル系で見て「平均重力の向き」に 0.227 mm 動きます。

しかしこの「平均重力の向き」は、内側フレームの回転で位相 (β₀) が変わるので、20分かけて 1周します。1周の間の動きを足し算 (積分) すると:

```
20分後の位置変化 = 内側フレーム1周ぶんの動きの合計
                = 円1周ぶんの動き
                = 出発点に戻る (1周したので)
```

つまり 20分後、細胞はもとの位置に戻ります。途中で円を描いて移動しますが、サンプル容器の中で同じ場所に戻ってきます。

### 8.3 1周の間で細胞はどこまで遠くに行くか

途中で一番遠くに行くのは半周した時点 (10分後) で、その距離は次の式で表せます。

```
最大距離 = 沈降速度 × 目標重力比 × 位相ドリフト周期 ÷ π
        = 1 μm/秒 × 0.378 × 1200秒 ÷ π
        = 144 μm = 0.144 mm
```

このとき、サンプル容器の半径が 5 mm = 5000 μm なので、最大距離 144 μm はサンプル容器の半径の 144/5000 ≈ 2.9% でしかありません。つまり細胞は容器の壁にまで届かず、容器の中心付近で小さな円を描くだけです。

ツールが出力する「ドリフト振幅 0.144 mm」とこの計算は完全に一致します。

### 8.4 沈殿が成立しない条件

最大距離 < 容器の半径 を満たせば沈殿は起こりません。式にすると:

```
沈降速度 × 目標重力比 × 位相ドリフト周期 ÷ π  <  容器の半径
```

両辺整理すると:

```
位相ドリフト周期  <  π × 容器の半径 ÷ (沈降速度 × 目標重力比)
```

細菌 (沈降速度 1 μm/秒)、容器半径 5 mm、火星重力 (0.378) で計算すると:

```
位相ドリフト周期  <  π × 5000 ÷ (1 × 0.378)
                ≈  41,500 秒
                ≈  11.5 時間
```

なので位相ドリフト周期 20分は条件を余裕で満たしています。

### 8.5 24-48時間の長時間培養での評価

実際の細菌培養は **24-48時間** の長時間運用を想定しています。1周期 (20分) で細胞が出発点に戻るという §8.2 の結果は、何周しても繰り返されるので、原理的には何時間運用しても累積的な沈殿は起きません。培養時間を時間単位で換算すると:

- 24時間 = 1440 分 = **72 周期** (位相ドリフト周期 20分の場合)
- 48時間 = 2880 分 = **144 周期**

ただし「理論上閉じた軌道」と「実機での挙動」には少し差があるので、長時間運用時に気を付けるべきことを以下に整理します。

#### 8.5.1 理論的累積誤差 (モーター精度起因)

実機ではモーターの回転速度に微小な誤差 (±0.1% 程度) があり、1周期ごとに細胞の出発点がわずかにずれます。1周期あたりの誤差を概算で 1-10 μm とすると:

- 24時間後の累積誤差: 72周期 × 10 μm = 720 μm = **0.72 mm**
- 48時間後の累積誤差: 144周期 × 10 μm = 1440 μm = **1.44 mm**

これらはチャンバ半径 5 mm の 30% 以下に収まります。

#### 8.5.2 熱拡散 (ブラウン運動) の寄与

細菌は重力以外にもブラウン運動でランダムに動きます。大腸菌の拡散係数を D ≈ 1 × 10⁻⁹ m²/秒 と仮定すると、時間 t での拡散距離は √(D × t) で計算できます:

| 培養時間 | 拡散距離 √(D·t) | チャンバ半径 5mm との比 |
|----------|-----------------|------------------------|
| 1時間    | 60 μm          | 1.2% |
| 24時間   | 290 μm         | 5.8% |
| 48時間   | 410 μm         | 8.2% |

拡散はランダムなので「沈殿させる方向」を持たず、むしろ細胞を容器内で均一化する方向に働きます。長時間培養では、ドリフト振幅 (144 μm) と拡散距離 (24h: 290 μm) が同オーダーなので、**拡散による均一化が支配的** になります。

#### 8.5.3 24-48時間運用での総合評価

各効果を足し合わせた、細胞の中心位置からの平均的なずれ:

| 培養時間 | ドリフト振幅 (理論) | 累積誤差 (モーター) | 拡散距離 | 合計 | チャンバ占有率 |
|----------|-------------------|--------------------|---------|---------|----------------|
| 1時間    | 0.144 mm          | 0.03 mm            | 0.06 mm | ~0.2 mm | 4% |
| 24時間   | 0.144 mm (閉軌道) | 0.72 mm            | 0.29 mm | ~1.2 mm | 24% |
| 48時間   | 0.144 mm          | 1.44 mm            | 0.41 mm | ~2.0 mm | 40% |

**結論**: 48時間運用でもチャンバ半径の 40% 以内に細胞分布が収まり、壁への沈殿到達は起きません。24-48時間の細菌培養における部分重力シミュレーションとして適切に動作します。

#### 8.5.4 長時間運用での実用上の注意

理論とは別に、実際の培養で考慮すべき点:

- **モーター発熱**: 連続48時間運転でモーターと駆動系が温度上昇する。トルクと回転精度に影響しうるので、放熱対策またはサーマルマージンを確保
- **培養液のガス交換**: 密閉容器だと O₂ 枯渇 / CO₂ 蓄積が起こる。長時間培養では半透膜 or 通気口の確保が必要
- **培養液の対流**: 培養液中の局所温度差で対流が起きると、細胞分布の均一化が早まる一方、重力認識を撹乱する可能性。サーモスタットで容器全体を一定温度に保つ
- **栄養枯渇 / 代謝物蓄積**: 細菌密度が増すと栄養が局所枯渇する。培地体積を十分大きく取るか、灌流系を組む
- **位相ドリフト周期の安定性**: T_drift が ±1秒以内の精度で安定していること。長時間で周期がずれると累積誤差が大きくなる

これらは本レポートの理論モデルでは扱わない「装置運用と培養生物学の問題」ですが、長時間運用の成否を左右するので付記しておきます。

## 10. 前提条件と適用範囲

このレポートが正しく成り立つには、いくつかの前提があります。

### 10.1 物理的な前提

1. **サンプルは 2軸の交点に置く**: もし軸からずれた位置にサンプルを置くと、回転による遠心力が混入して目標重力が変わってしまいます。
2. **回転は剛体的**: 装置全体が安定して回り続けていることが必要。回転が振動的だと前提が崩れます。
3. **流体の流れの影響は無視**: 培養液の中の流れ (容器内の渦など) が細胞に与える力は無視しています。実際の細胞培養ではこの効果も考慮が必要です。
4. **沈降速度の式 (ストークスの式) が成り立つ**: 細胞のサイズが小さく、培養液の粘性が支配的であることが前提。大きな細胞凝集塊や、培養液が非ニュートン流体である場合は別解析が必要です。

### 10.2 時間スケールの条件

各時間が次の順番に並んでいる必要があります。

```
外側フレームの 1回転の時間  <<  細胞応答時間  <<  位相ドリフト周期  <<  培養時間
(12秒)                       (約10分)        (約20分)             (24-48 時間)
```

(72-144 周期ぶんの動作になります。詳細は §8.5)

ツールはこの条件が満たされない設定を入力すると警告を出します。

### 10.3 適用できない場合

- **目標重力が 1.0 g より大きい**: 地球重力以上は本モードでは作れません。傾斜角 ψ を負にできないからです。
- **目標重力が 1.0 g に非常に近い**: 傾斜角がほぼゼロになり、外側フレームを回転させる意味がなくなります (ふつうに静置すればよい)。
- **粒子のサイズが大きすぎる**: 細胞が容器サイズに比べて大きい場合、ストークスの仮定が崩れます。
- **真空中・低粘度の液体**: 細胞の沈降速度が大きくなり、位相ドリフト周期の許容上限が短くなります。

| 確認項目 | 結果 |
|----------|------|
| 細胞が感じる重力の大きさ | 短時間平均で正確に g₀ × cos ψ = 目標重力に一致する |
| 沈殿の防止 (短期) | 位相ドリフト周期内で細胞は円を描いて元の位置に戻り、長時間平均では動きがない |

## 12. Centrifuge (遠心) モードの動作原理

ここまでは Tilt モードを扱ってきました。次は、**サンプルを内側フレームの 4角に置いて、内側回転の遠心力で重力を作る** Centrifuge モードについて整理します。

### 12.1 何が違うか

| 項目 | Tilt モード | Centrifuge モード |
|------|------------|-----------------|
| サンプル位置 | 2軸の交点 (r = 0) | 内軸から半径 8.5 cm (内側フレーム4角) |
| 重力の源 | 地球重力の射影 (傾斜) | 内側回転の遠心力 |
| 重力の式 | g₀ × cos ψ | ω_in² × r |
| 内側フレームの役目 | 重力方向をゆっくり1周 (沈殿防止) | 遠心力で g_target を生成 (高速回転) |
| 外側フレームの役目 | 傾斜軸まわりに高速回転 (横成分を均す) | 地球重力を時間平均で消す (クリノローテーション) |
| 沈殿 | なし (短時間平均で目標g、長時間平均でゼロ) | あり (遠心方向に細胞が動く) |

### 12.2 重力の式と計算

サンプルが内側軸から半径 r の位置にあるとき、内側フレームを角速度 ω_in で回すと遠心力が発生します。

```
遠心加速度 = ω_in² × r
```

これを目標重力に等しく設定:

```
ω_in² × r = g_target × g₀
   ⇒ ω_in = √(g_target × g₀ / r)
```

r = 0.085 m (8.5cm) で例:

| 目標重力 | ω_in 必要 |
|----------|----------|
| 1.0g | 102.6 RPM |
| 0.5g | 72.6 RPM |
| 0.378g (火星) | 63.1 RPM |
| 0.166g (月) | 41.8 RPM |
| 0.01g | 10.3 RPM |

### 12.3 外側フレームの役目: 地球重力の打ち消し

外側フレームを ω_out で(内側軸とは別の軸まわりに)回すと、サンプルから見て地球重力ベクトルがゆっくり回転します。十分な時間平均すれば、地球重力成分は時間平均でゼロになります。これがクリノローテーション。

ただし外側位置 (フレーム半径 R_outer = 14.5 cm) でも遠心力が発生するので、これを目標重力の 10% 以下に抑える必要があります:

```
ω_out² × R_outer ≪ g_target × g₀
⇒ ω_out ≪ √(g_target × g₀ / R_outer)
```

火星 (g_target = 0.378) で:

```
ω_out ≪ √(0.378 × 9.81 / 0.145) = 5.06 rad/s ≈ 48 RPM
```

実用上 ω_out = 5 RPM 程度に設定すれば外乱は 0.004g 程度に収まります。

### 12.4 沈殿の不可避性

§11 とは違い、**遠心モードでは沈殿が必ず発生します**。理由は §11 で書いたとおり: サンプル本体に固定された遠心力ベクトルがあり、それは回転を組み合わせても消せないからです。

サンプル系で見た沈降速度:

```
v_sed_effective = v_sed × g_target
```

例: バクテリア (v_sed = 1 μm/秒)、火星 (g_target = 0.378):

```
v_sed_effective = 1 × 0.378 = 0.378 μm/秒
```

チャンバ半径 5 mm = 5000 μm の容器で、細胞が壁に到達する時間:

```
壁到達時間 = チャンバ半径 ÷ 有効沈降速度
          = 5000 ÷ 0.378
          = 13,228 秒 ≈ 3.67 時間
```

つまり 24 時間培養なら、最初の 3.67 時間で細胞は遠心方向の壁に到達し、残り 20.33 時間 (84.7%) は壁に張り付いた状態で過ごします。

### 12.5 これは「正常な挙動」か?

物理的にはまったく正常で、Mars/Moon 表面の実環境でも同じことが起きます。火星の細菌培養容器でも、細胞は 0.378g の重力で 0.378 μm/秒で沈降して、最終的に容器底に層を作ります。

→ **「Mars analog 実験」 (火星環境の再現)** としては Centrifuge モードは正しい挙動を再現
→ **「火星重力下で懸濁状態を維持して観察」** という目的には不向き (それは Tilt モードを使う)

### 12.6 各 g_target の Centrifuge モード設定

ツールが返す値の一覧 (r_sample = 8.5cm, ω_out = 5 RPM, チャンバ半径 5 mm, 培養 24h):

| 目標重力 | ω_in 内側 | ω_out 外側 | 沈降速度 | 壁到達時間 | 壁滞在比率 |
|----------|-----------|-----------|---------|-----------|----------|
| 1.0g (地球) | 102.6 RPM | 5 RPM | 1.0 μm/s | 1.39 h | 94.2% |
| 0.378g (火星) | 63.1 RPM | 5 RPM | 0.38 μm/s | 3.67 h | 84.7% |
| 0.166g (月) | 41.8 RPM | 5 RPM | 0.17 μm/s | 8.37 h | 65.1% |
| 0.01g | 10.3 RPM | 5 RPM | 0.01 μm/s | 138.9 h | 0% |

g_target が小さいほど沈降が遅く、壁に到達するまでに時間がかかります。

## 13. Hybrid モード (任意のサンプル半径)

サンプルが「完全に中央 (r=0)」でも「4角 (r=8.5cm)」でもなく、その中間 (例: 2 cm オフセット) にある場合のハイブリッドモード。

### 13.1 サンプル系での重力ベクトルの合成

サンプルが軸交点から距離 r 離れているとき、サンプル本体系で見える重力は **2つの寄与の合成** になります。

```
g_tilt        = g0 × cos ψ        ← 地球重力の射影 (ゆっくり方向ドリフトする)
g_centrifugal = ω_in² × r          ← 内側回転の遠心力 (body に固定された方向)
```

§11 で示したとおり、遠心力ベクトルはサンプル本体に固定された方向を向き、tilt 寄与のベクトルはサンプル本体系内をゆっくり回転します。両者の方向は **互いに直交** するので (tilt は q̂・n̂_out 平面内、centrifugal は x̂_body 方向)、合成は二乗和:

```
|⟨g⟩| = √( (g0·cos ψ)² + (ω_in² × r)² )
      = √( g_tilt² + g_centrifugal² )
```

これを g_target × g₀ に等しくすると:

```
g_target² × g0² = g_tilt² + g_centrifugal²
              ⇒ g_tilt = √(g_target² × g0² - g_centrifugal²)
              ⇒ ψ = arccos( g_tilt / g0 )
                  = arccos( √(g_target² - (ω_in²·r/g0)²) )
```

### 13.2 ドリフト用 ω_in での遠心寄与は微小

Hybrid モードでは Tilt モードと同じく ω_in を超低速 (位相ドリフト用) に固定します。典型値 ω_in = 0.05 RPM ≈ 5.24 × 10⁻³ rad/秒 で計算すると:

| サンプル半径 r | ω_in² × r [m/s²] | g 単位 |
|--------------|------------------|--------|
| 0.5 cm   | 1.37 × 10⁻⁷ | 1.40 × 10⁻⁸ g |
| 1 cm     | 2.74 × 10⁻⁷ | 2.80 × 10⁻⁸ g |
| 2 cm     | 5.48 × 10⁻⁷ | 5.59 × 10⁻⁸ g |
| 5 cm     | 1.37 × 10⁻⁶ | 1.40 × 10⁻⁷ g |
| 8.5 cm   | 2.33 × 10⁻⁶ | 2.37 × 10⁻⁷ g |

→ 数 cm のオフセットでは遠心寄与は **10⁻⁷ g オーダー** で、火星 (0.378 g) や月 (0.166 g) の目標と比較して完全に無視できる小ささ。実用上、ψ は通常の tilt モード (arccos(g_target)) と区別がつかない。

### 13.3 遠心由来の累積沈降

ただし遠心寄与は body に固定された方向を向くので、振動ではなく **時間に比例して累積する** 沈降を生みます:

```
累積変位 = v_sed × g_centrifugal × 培養時間
       = v_sed × (ω_in²·r/g0) × T
```

例: バクテリア (v_sed = 1 μm/秒)、r = 2 cm、48 時間培養:

```
累積変位 = 1e-6 × 5.59e-8 × 172800 = 9.7e-9 m = 0.0097 μm
```

完全に無視できる。10 cm オフセットでも 48 時間で 0.05 mm 程度。

### 13.4 Hybrid モードが破綻する条件

遠心寄与が大きくなりすぎると、tilt 寄与だけで g_target を補えなくなる。具体的には:

```
g_centrifugal > g_target × g0   ⇒   ω_in²·r > g_target × g0
```

これが起きたら hybrid モードでは扱えず、centrifuge モードに切り替えるべき。

典型ドリフト ω_in = 0.05 RPM では、g_target = 0.378 g に対して破綻するのは:

```
r > 0.378 × 9.81 / (5.24e-3)² ≈ 1.35 × 10⁸ m   ← 物理的にあり得ない
```

つまり、Tilt モード相当の低速 ω_in を使う限り、任意の r で hybrid モードは機能する。

### 13.5 ツールでの使い方

```bash
# 軸交点から 2 cm オフセットのサンプルで火星重力
python3 clinostat.py 0.378 --mode hybrid --sample-r-cm 2

# 5 cm オフセット、月重力、48時間培養
python3 clinostat.py 0.166 --mode hybrid --sample-r-cm 5 --culture-hours 48
```

出力で `g_cent` (遠心由来の寄与) と `合計変位` (tilt ドリフト + 遠心累積) を確認できる。

→ **2軸交点から多少ずれていても tilt モード相当の効果が得られる、と理論的に保証される**。

## 14. 4つのモードの比較とフレーム半径の意味

### 14.1 サンプル位置とフレーム半径の関係

「フレーム半径 8.5cm / 14.5cm」がどう効くかは **サンプル位置によって変わります**:

- **Tilt / Switching モード (サンプル=2軸交点 r≈0)**:
  フレーム半径は **重力の計算式に登場しません**。装置の構造寸法 (どれくらいの大きさの機械か) というだけの意味。回転速度を決めるのは「位相ドリフト周期」と「目標重力」のみ。

- **Centrifuge モード (サンプル=4角 r=8.5cm)**:
  内側フレーム半径 8.5cm が **遠心力の式に直接登場** します (`ω_in² × 8.5cm = g_target × g₀`)。外側フレーム半径 14.5cm は「外側位置での遠心外乱」を評価するのに使います。

- **Hybrid モード (サンプル=任意の r)**:
  ユーザが r を指定。tilt 寄与と遠心寄与の **両方** が登場するが、ω_in をドリフト用に低速設定するので遠心寄与は通常微小 (§13.2)。

つまり中央配置 (Tilt/Switching) では、フレーム半径の機械寸法的な意味は残るが、重力計算には影響しなくなる、という整理です。Hybrid では r が連続パラメータとして登場します。

### 14.2 モード選択早見表

| 実験目的 | 推奨モード | 理由 |
|----------|-----------|------|
| 火星/月の表面環境を細胞に再現したい (Mars analog) | Centrifuge | 部分重力 + 沈殿の両方を再現できる |
| 部分重力下で細胞を懸濁状態で観察したい | Tilt | 沈殿なしで g_target を維持できる |
| サンプルを完全に軸交点には置けないが懸濁したい | Hybrid | 数 cm のオフセットを許容しつつ tilt 相当の性能 |
| 重力を 0g に近い領域でかけたい | Tilt または Switching | 両方とも微小重力近傍をカバー |
| ハードを傾斜できない (傾斜マウントが作れない) | Switching | 上限 0.5g だが傾斜不要 |
| 1g 超の過重力 (hypergravity) | Centrifuge | ω_in を上げれば 1g 以上も可能 |

### 14.3 同じ g_target を 4 モードで作ったときの設定値

火星 (g_target = 0.378) を例にして、各モードの設定値を並べます:

| 設定項目 | Tilt | Hybrid (r=2cm) | Centrifuge (r=8.5cm) | Switching |
|----------|------|----------------|---------------------|----------|
| サンプル位置 | 2軸交点 | 軸交点から 2 cm | 内軸から 8.5 cm | 2軸交点 |
| 傾斜角 ψ | 67.79° | 67.79° (実質同じ) | (関係なし) | 0° (傾斜なし) |
| ω_in 内側 | 0.05 RPM | 0.05 RPM | **63.1 RPM** (高速) | 5.05 RPM ↔ 3.09 RPM |
| ω_out 外側 | 5 RPM | 5 RPM | 5 RPM | 5 RPM |
| 沈殿 | なし (0.14 mm 振動) | なし (0.14 mm 振動 + 10⁻⁵ mm 累積) | あり (3.67 h で壁) | なし (0.19 mm 振動) |
| 達成可能上限 | 1.0 g | 1.0 g | 1.0 g 以上も可 | 0.5 g |

→ Hybrid と Tilt は r が小さければ実質同じ結果。サンプル位置のずれを許容できる柔軟性が hybrid の強み。

### 14.4 ツールでの切替

```bash
# Tilt モード (既定, 推奨: 懸濁培養, サンプル=軸交点)
python3 clinostat.py 0.378

# Hybrid モード (サンプル=任意のr, 例: 2cmオフセット)
python3 clinostat.py 0.378 --mode hybrid --sample-r-cm 2

# Centrifuge モード (Mars analog 実験, サンプル=4角)
python3 clinostat.py 0.378 --mode centrifuge

# Switching モード (傾斜なし運用)
python3 clinostat.py 0.378 --mode switching

# 全モード並べて表示
python3 clinostat.py 0.378 --mode all
```

Hybrid / Centrifuge では培養時間とサンプル位置を指定可能:

```bash
python3 clinostat.py 0.378 --mode hybrid --sample-r-cm 2 --culture-hours 48
python3 clinostat.py 0.378 --mode centrifuge --sample-r-cm 8.5 --culture-hours 48
```

## 付録. 特殊な設定での確認

理論が極端な場合でも正しく動くか確かめます。

- **傾斜角 0° (装置を立てたまま)**: cos 0° = 1 なので、サンプルが感じる重力は g₀ × 1 = 9.81 m/秒² = そのまま地球重力。回転させなくても結果は同じで、当然の結果。
- **傾斜角 90° (装置を寝かせる)**: cos 90° = 0 なので、サンプルが感じる重力は 0。これは無重力シミュレーション。
- **傾斜角 60° (装置を斜めにする)**: cos 60° = 0.5 なので、サンプルが感じる重力は 9.81 × 0.5 = 4.905 m/秒² = 地球の半分の重力。
- **内側フレームを止めた状態 (角速度 = 0)**: 重力の向きは時間で1周しないので、細胞は一方向にずっと沈み続け、沈殿が起きる。なので **内側フレームの低速回転は必須**。
- **内側フレームを速くしすぎた状態**: 重力の向きが速く変わりすぎて、細胞は重力の強さを「ゼロ」と感じてしまう (微小重力シミュレーションになってしまう)。なので **内側は十分ゆっくり** にする必要がある。

これらは §4-§7 の主要な結論と整合し、解析が間違っていないことを別の角度から確認しています。
